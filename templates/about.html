About Talent Trail

Helping you discover the right career path

What is Talent Trail?

Talent Trail is a web-based career exploration tool designed to help users discover careers that align with their skills and abilities. The application compares user input with real-world job requirements and provides a compatibility score.

How it Works

  1. Select a career you are interested in
  2. Take the skills quiz
  3. Receive a compatibility score
  4. Identify matched skills and areas for improvement

Why We Built This

Choosing a career can be overwhelming. Talent Trail simplifies the process by giving users clear, data-driven feedback about where they stand and how they can improve.

How the Algorithm Works

When the user picks a career, the app pulls the top skills for that job based on ONET importance, so the survey is only showing the skills that matter most for that role. Then, after the user rates themselves on those skills, the app compares the user’s rating to the ONET level rating for each skill. It calculates the gap by subtracting the user’s level from the career’s target level, and if the user is already at or above that level, the gap is just treated as zero.

The algorithm weights the gap by the importance score, so skills that are both important as defined by ONET data AND lacking from the user input get ranked higher. The final results show which important skills the user already matches and which ones they still need to improve, with the improvement list ordered by the biggest weighted gaps first.

What is ONET?

ONET, which stands for the Occupational Information Network, is the nation’s primary source of occupational and career information. ONET data is job and workforce data pulled directly from the O*NET database/website, and it includes standardized information about occupations such as skills, knowledge, abilities, work activities, and ratings like Importance and Level for each skill associated with a job.
